Telegram бот для управления проектами. Позволяет создавать, редактировать и отслеживать проекты, добавлять навыки и управлять статусами выполнения.

## 📋 Возможности бота

- **Создание проектов** - добавление нового проекта с названием, описанием, ссылкой и статусом
- **Просмотр проектов** - отображение списка всех проектов пользователя
- **Детальная информация** - просмотр полной информации о конкретном проекте
- **Редактирование проектов** - изменение названия, описания, ссылки или статуса проекта
- **Управление навыками** - добавление навыков к проектам (Python, SQL, API, Telegram)
- **Удаление проектов** - возможность удалить ненужные проекты

## 🛠 Технологии

- Python 3.7+
- python-telegram-bot
- SQLite3
